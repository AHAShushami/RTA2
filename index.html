📐 Statistical Methods

The following analytical tests are performed on the RTA dataset:

  • Chi-Square Test (χ²) – Tests association between categorical variables (gender, vehicle type, injury severity, year).
  • Cochran-Armitage Trend Test – Tests for significant linear trend in RTA proportion across years.
  • Fisher's Exact Test – Used when expected cell counts <5.
  • Odds Ratio (OR) – Measures strength of association between exposure and outcome.
  • Binary Logistic Regression – Multivariate analysis of injury severity predictors.
  • Kruskal-Wallis Test – Compares MC days across groups.

Significance level: α = 0.05. All tests two-tailed.

📖 Content Analysis Methodology

We use Thematic analysis following Braun & Clarke's (2006) 6-phase reflexive framework was applied to three free-text fields:

  • Ringkasan Kejadian (Incident Narrative) – n=0 responses
  • Faktor Penyebab Utama (Primary Causative Factor) – n=0 responses
  • Faktor Penyumbang (Contributing Factors) – n=0 responses

Phases: 1) Familiarization → 2) Initial Coding → 3) Theme Search → 4) Theme Review → 5) Theme Definition → 6) Reporting
